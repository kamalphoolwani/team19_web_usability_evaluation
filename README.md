## Prerequisites and Installation
```
Install NodeJs from NodeJs Official Page.
Open Terminal
Go to your file project
Run in terminal: npm install -g @angular/cli
```

## To Run locally:
```
* npm install
* ng serve
Navigate to: http://localhost:4200/
```
